# Examples

This directory will hold worked examples of anchorfile in real codebases, showing the full workflow: source asset → annotated sidecar → generated or runtime-loaded code → correct UI.

## Planned examples

- **Game level map** — an illustrated level with spawn points, treasure chest hitboxes, and circular trigger zones, consumed by a game engine or framework like Unity, Godot, or Bevy.
- **Illustrated app background** — a painted UI backdrop with tap targets layered on top, consumed by a mobile app (SwiftUI, Compose, Flutter).
- **HTML image map** — a diagram with clickable regions, consumed as `<area>` elements or CSS `clip-path` values.
- **PDF form placement** — form fields overlaid on a scanned page, consumed by a form renderer.

## Contributing an example

The most valuable examples are ones that show a painful before (hardcoded coordinates, manual measurement scripts, eyeballing in an image editor) and a clean after. When the tooling is further along, contribution guidelines for examples will land here.
